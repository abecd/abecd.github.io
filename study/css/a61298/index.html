# CSS背景

  1. 背景颜色(background-color)

  2. 背景图片(background-image)

    • 注意背景颜色和背景图片可以同时设置,背景色会在背景图片的下方

    • 如果图片小就平铺,图片大就隐藏

  3. 背景图片重复方式(background-repeat)

    • repeat(默认值)
    • repeat-x(沿着x轴重复)
    • repeat-y(沿着y轴重复)
    • no-repeat(不重复)

  4. 设置背景图片位置(background-position)

    • 可以使用top/left/center/right/bottom其中的两个值(水平/垂直)来设置图片的位置,如果只写一个第二个值默认为center。
    • 也可以用两个偏移量(水平/垂直)来指定位置

  5. 设置背景的裁切(background-clip)

    • border-box(默认值):会裁切内容区,内边距,边框内容
    • padding-box:会裁切在内容区,内边距的内容
    • content-box:会裁切在内容区的内容

  6. 设置背景图片偏移(background-position)的原点(background-origin)

    • padding-box(默认值):在内边距的左上角
    • border-box:在边框的左上角
    • content-box:在内容区的左上角

  7. 设置背景图片的大小(background-size)

    • 设置两个值宽度/高度
    • 如果只设置一个,第二个就是auto。
    • 一些特殊的值
      • cover:图片的比例不变,将元素铺满元素
      • contain:图片比例不变,将图片完整显示[默认]

  8. 设置背景图片是否随视窗(viewpoint)滚动条滚动(background-attachment)

    • scroll(默认值):视窗随滚动条滚动
    • fixed:不随视窗滚动条滚动并且和position:fixed相似偏移原点在视窗的左上角且无法改变。

  9. 简写属性(background)要求

    • background-size必须写在background-positon后用/隔开
    • background-clip必须在background-origin的后面
    • 一些背景的细节:
      • 4,6,8项之间存在关联

  10. 图片的加载

    • 图片属于网页中的外部资源,外部资源都需要浏览器去请求按需加载。
    • 所以在一些图片的切换时可能会因为网络问题导致加载间隙有空白期。

  11. CSS-sprite : 精灵图/雪碧图

    • 很多小图构成的一个大图,用作背景图设置
    • 特点:一次性将多个图片加载进页面，降低请求的次数，加快访问速度，提升用户体验。

  12. 渐变(渐变是图片需要通过background-image来设置)

    • 线性渐变

      • background-image:liner-gradient([dir],start [distance],...,end[distance]);[高度中非渐变部分用纯色填充]/background-image:repeating-liner-graient(dir,start[distance],...,end [distance])[高度中非渐变部分重复渐变来填充]
      • dir:xxxdeg,to left/to right/to bottom/to top distance:纯色开始渐变距离
      • 默认从上向下

    • 径向渐变

      • 从中心点向四处放射
      • radical-gradient([[a,b(横向半径,纵向半径)] [at c,d(圆心位置)]],start [distance],...,end [distance]])/repeating-radical-gradient([[a,b(横向半径,纵向半径)] [at c,d(圆心位置)]],start [distance],...,end [distance]])
